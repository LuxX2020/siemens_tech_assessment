# 变量表格编辑器

这是一个用于管理和编辑变量定义的工具，支持可视化编辑、标准文本格式导入导出以及数据类型校验。

## 功能特性

- **可视化编辑**：支持添加、删除、修改变量行。
- **自动索引**：Index 列自动维护，删除行后自动重排。
- **数据校验**：
    - 支持的数据类型：`BOOL`, `INT`。
    - 名称唯一性校验（不区分大小写）。
    - INT 类型范围校验 (-2147483648 到 2147483647)。
- **智能联动**：切换数据类型时，自动重置默认值（BOOL -> TRUE, INT -> 0）。
- **导入导出**：支持标准 PLC 变量定义格式（VAR...END_VAR）的文本导入与导出。
